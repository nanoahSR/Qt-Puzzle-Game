# QT-拼图游戏

## 环境

QtCreator、C++11

## 开发过程

技术上没啥特别的。

导入图片，分为9块。

用一个map存储“图片块”，初始化按位置顺序编号。

点击“开始游戏”打乱9张图片的位置，按照编号重新显示相应的图片。

每移动一次就遍历map有没有恢复顺序。

循环，直至复位。

源码：[nanoahSR/Qt-Puzzle-Game: 使用Qt做的简单拼图游戏，玩法类似于【华容道】 (github.com)](https://github.com/nanoahSR/Qt-Puzzle-Game)
